# Fireworks Gathering Landing Page

A simple public landing page for sharing details about the July 3 fireworks gathering.

## Purpose

This site is meant to be the single easy-to-share place for:

- Date, time, and weather updates
- Parking and viewing instructions
- What guests should bring
- Optional contribution link
- Photos and event details

## First version content plan

- Event status banner: `On as planned` / weather update / rain date
- Date: July 3
- Fireworks: after sundown, approximately 10:00 PM
- Bring your own chair, blanket, food, and drinks
- Guests may bring a few others within reason
- Parking and viewing maps/photos to be added
- Venmo contribution link to help offset show costs

## Development

This repo will hold a simple static landing page that can be hosted with GitHub Pages.
